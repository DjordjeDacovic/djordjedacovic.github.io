## 🛠️ How to Run the Scripts

After downloading the scripts, you’ll want to run them. Here’s how:

### For Windows Users:

1. **Locate the Downloaded Files:** Open the folder where you saved the scripts.
2. **Extract Files:** If the files are in a ZIP folder, right-click the folder and select “Extract All.”
3. **Open Command Prompt:** Press `Windows + R`, type `cmd`, and hit `Enter`.
4. **Navigate to the Folder:** Use the `cd` command to go to the folder where you extracted the scripts. For example:
   ```
   cd C:\path\to\your\folder
   ```
5. **Run a Script:** Type the name of the script and press `Enter.`

### For macOS Users:

1. **Locate Your Files:** Open Finder and find the folder with your downloaded scripts.
2. **Extract Files:** If in a ZIP file, double-click the ZIP to extract it.
3. **Open Terminal:** Press `Command + Space`, type `Terminal`, and hit `Enter`.
4. **Navigate to the Folder:** Use the `cd` command to go to the folder. For example:
   ```
   cd /Users/yourusername/path/to/your/folder
   ```
5. **Run a Script:** Type the script name and press `Enter.`

### For Linux Users:

1. **Locate Your Files:** Open your file manager and find the downloaded scripts.
2. **Extract Files:** If they're in a ZIP format, right-click and choose "Extract Here."
3. **Open Terminal:** Press `Ctrl + Alt + T`.
4. **Navigate to the Folder:** Use the `cd` command to get to the folder. For example:
   ```
   cd /home/yourusername/path/to/your/folder
   ```
5. **Run a Script:** Type the script name followed by `./` and hit `Enter`.
